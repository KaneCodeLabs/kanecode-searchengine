How to install

jsDelivr

Paste the following code into the <head> section:

<link href="https://cdn.jsdelivr.net/gh/KaneCodeLabs/kanecode-searchengine@1.0.0/dist/searchengine.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/gh/KaneCodeLabs/kanecode-searchengine@1.0.0/dist/searchengine.min.js"></script>
GitHub

Step 1

Download the latest version of the Kanecode Search Engine from GitHub.

Step 2

Copy the contents of the kanecode-searchengine folder into your project.

Step 3

Paste the following code into the <head> section:

<link href="<your-project-path>/kanecode-searchengine/dist/searchengine.min.css" rel="stylesheet">
<script src="<your-project-path>/kanecode-searchengine/dist/searchengine.min.js"></script>
npm (upcoming)

How to use

To use the search engine, you need to create an empty element and call a script to mount the search engine into it. The following code snippet will do this:

<div class="search-selector"></div>

<script>
// The data to be searched
var fruits = ['Apple', 'Orange', 'Pear', 'Banana', 'Strawberry'];

// The element to mount the search engine into
var selector = document.querySelector('.search-selector');

// Initialization and configuration of the search engine
var search = new KCSearchEngine(selector, {
	data: fruits
});
</script>

Introduction

Introduction

This is a search engine that can be used to search data from any source. The files included in this library mount a search bar with recommendations and is fully configurable.

First steps

To use this library you can use the following code to implement it in your HTML file.

Includes the following lines to <head> section:

<link href="https://cdn.jsdelivr.net/gh/KaneCodeLabs/kanecode-searchengine/dist/searchengine.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/gh/KaneCodeLabs/kanecode-searchengine/dist/searchengine.min.js"></script>

Includes the following lines to <body> section:

<div class="search-bar"></div>
<script>
	new KCSearchengine(document.querySelector('.search-bar'), {
		data: ['Apple', 'Orange', 'Strawberry']
	});
</script>

Installation

How to install

jsDelivr

Paste the following code into the <head> section:

<link href="https://cdn.jsdelivr.net/gh/KaneCodeLabs/kanecode-searchengine/dist/searchengine.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/gh/KaneCodeLabs/kanecode-searchengine/dist/searchengine.min.js"></script>
GitHub

Step 1

Download the latest version of the Kanecode Search Engine from GitHub.

Step 2

Copy the contents of the kanecode-searchengine folder into your project.

Step 3

Paste the following code into the <head> section:

<link href="<your-project-path>/kanecode-searchengine/dist/searchengine.min.css" rel="stylesheet">
<script src="<your-project-path>/kanecode-searchengine/dist/searchengine.min.js"></script>

How to use

To use the search engine, you need to create an empty element and call a script to mount the search engine into it. The following code snippet will do this:

<div class="search-selector"></div>

<script>
	// The data to be searched
	var fruits = ['Apple', 'Orange', 'Pear', 'Banana', 'Strawberry'];

	// The element to mount the search engine into
	var selector = document.querySelector('.search-selector');

	// Initialization and configuration of the search engine
	var search = new KCSearchEngine(selector, {
		data: fruits
	});
</script>
